# Personal Task Manager

A simple and efficient task management application built with C++ and Qt.

## Features
- Add, edit, delete tasks
- Categorize tasks
- Notifications for task deadlines
- Save and load tasks from a file

## Setup
- Clone the repository and build using Qt Creator.

## Usage
- Run the application to manage your tasks.
